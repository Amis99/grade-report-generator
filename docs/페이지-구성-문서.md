# 성적표 생성 프로그램 - 페이지 구성 문서

## 1. 관리자 페이지 (index.html)

### 1.1 헤더
| 구성 요소 | 설명 | 웹/모바일 |
|----------|------|-----------|
| 로고 (images/logo.png) | 국어농장 로고 이미지 | 공통 |
| 타이틀 "성적표 생성 프로그램" | 앱 제목 | 웹만 (모바일에서 숨김) |
| 사용자 정보 | 이름, 소속 표시 | 웹만 |
| 햄버거 메뉴 버튼 | 드롭다운 메뉴 열기 | 공통 |

**헤더 드롭다운 메뉴:**
- 학생 계정 관리
- 관리자 메뉴 (대기 중인 가입 수 배지)
- 데이터 백업
- 데이터 복원
- 비밀번호 변경
- 로그아웃

---

### 1.2 탭 네비게이션 (8개 탭)

| 탭 이름 | 탭 ID | 설명 |
|--------|-------|------|
| 시험 관리 | exam-management | 시험 CRUD, 문제 관리 |
| 학생 관리 | student-management | 학생 목록, 중복 감지 |
| 수강반 관리 | class-management | 수강반 CRUD |
| 과제 관리 | assignment-management | 과제 배정, 제출 관리 |
| 답안 입력 | answer-input | 학생 답안 입력 시트 |
| 채점 및 분석 | grading | 성적 분석, 문제별 통계 |
| 성적표 생성 | report | PDF 성적표 출력 |
| 학생별 오답 노트 | wrong-note | 오답 분석 PDF |
