1.1 로그인 방법

선생님께서 안내해 주신 주소로 접속하면 로그인 화면이 나타납니다. 학원에서 부여받은 아이디와 비밀번호를 입력하세요.

  1. 브라우저 주소창에 접속 주소를 입력합니다.
  2. 아이디(이름 또는 학원 ID)를 입력합니다.
  3. 비밀번호를 입력합니다.
  4. 로그인 버튼을 누릅니다.
💡 팁: 아이디나 비밀번호를 모를 때는 선생님께 문의하세요.
⚠️ 비밀번호를 여러 번 잘못 입력하면 빨간 오류 메시지가 표시됩니다.

1.2 첫 로그인 — 임시 비밀번호 변경

처음 로그인할 때는 선생님이 설정한 임시 비밀번호를 사용합니다. 로그인 후 반드시 본인만의 비밀번호로 변경해 주세요.

  1. 선생님께 받은 임시 비밀번호로 로그인합니다.
  2. 좌측 메뉴에서 ⚙️ 설정을 누릅니다.
  3. 🔑 비밀번호 변경 버튼을 누릅니다.
  4. 현재 비밀번호에 임시 비밀번호를 입력합니다.
  5. 새 비밀번호를 입력합니다.
  6. 새 비밀번호 확인에 동일하게 다시 입력합니다.
  7. 변경 버튼을 누르면 완료됩니다.
⚠️ 주의: 비밀번호는 본인만 알 수 있도록 관리하세요. 다른 사람에게 알려주지 마세요.
💡 비밀번호를 잊어버린 경우, 선생님께 비밀번호 초기화를 요청하세요.

2 대시보드

로그인하면 가장 먼저 보이는 화면입니다. 오늘의 할 일, 최근 성적, 선생님 코멘트를 한눈에 확인할 수 있습니다.

① 환영 메시지

상단에 "안녕하세요, [이름]님!" 메시지와 오늘 날짜가 표시됩니다. 상단 분홍색 바로 꾸며진 카드입니다.

② 📌 해야 할 것

아직 완료하지 않은 과제나 확인해야 할 성적표가 표시됩니다. 항목을 누르면 해당 페이지로 이동합니다.

③ 📊 최근 성적

가장 최근에 본 시험의 점수와 정보가 요약되어 카드로 나타납니다.

④ 💬 선생님 코멘트

선생님이 성적표에 남긴 코멘트를 확인할 수 있습니다.

  • 매일 확인: 로그인하면 대시보드에서 오늘 해야 할 일을 먼저 확인하세요.
  • 빠른 이동: 피드 카드를 누르면 해당 페이지로 바로 이동합니다.
  • 선생님 피드백: 코멘트 카드에서 선생님의 조언을 꼭 읽어보세요.
  • 새로운 항목: 새로 등록된 시험 결과나 과제가 있으면 상단에 표시됩니다.

4.2 과제 상세 및 페이지 제출

과제 카드를 누르면 상세 화면이 나타납니다. 각 페이지를 사진으로 찍어 제출합니다.

  1. 과제 목록에서 과제를 누릅니다.
  2. 페이지 그리드에서 제출할 페이지를 선택합니다.
  3. 카메라가 열리면 A4 가이드라인에 맞춰 촬영합니다.
  4. 촬영된 사진을 확인하고 제출을 누릅니다.
  5. 모든 페이지를 제출하면 과제가 완료됩니다.

페이지 그리드

각 페이지는 A4 비율(1:1.414)의 작은 카드로 표시됩니다. 카드 상태에 따라 테두리 색이 다릅니다.

📘 안내: 페이지를 눌러 촬영하거나 파일을 선택하세요. 한 번에 한 페이지씩 제출합니다.

4.3 카메라 촬영

페이지를 누르면 카메라 모달이 열립니다. 교재 페이지를 A4 가이드라인에 맞춰 촬영해 주세요.

촬영 방법

  1. 카메라 화면의 점선 사각형(A4 가이드라인)에 교재를 맞춥니다.
  2. 밝은 곳에서 흔들리지 않게 촬영합니다.
  3. 하단의 흰색 원형 버튼을 눌러 촬영합니다.
  4. 촬영된 이미지를 확인하고 제출합니다.

파일 선택 옵션

카메라 대신 갤러리에서 이미 찍어둔 사진을 선택할 수도 있습니다. 카메라 하단의 "파일 선택" 링크를 누르세요.

⚠️ 촬영 팁:
  • 밝은 조명 아래에서 촬영하세요
  • 그림자가 지지 않도록 주의하세요
  • 글씨가 선명하게 보이도록 초점을 맞추세요
  • 가이드라인 안에 교재가 모두 들어오게 하세요

4.4 페이지 상태 설명

과제의 각 페이지는 제출 및 검토 상태에 따라 다른 색상으로 표시됩니다.

상태 테두리 색 아이콘 설명
미제출 — 아직 제출하지 않은 페이지. 눌러서 촬영할 수 있습니다.
제출됨 ◷ 제출했지만 아직 선생님이 확인하지 않은 상태입니다.
완료 ✓ 선생님이 확인하여 승인한 페이지입니다.
거부됨 ✕ 사진이 불량하여 재촬영이 필요한 페이지입니다. 다시 제출하세요.
💡 거부된 페이지: 페이지를 다시 눌러 재촬영 후 제출하면 됩니다. 선생님의 피드백을 확인하고 더 선명하게 촬영해 보세요.

과제 제출 흐름 요약

미제출 → 촬영 → 제출됨 → 검토 → 완료 ✓
또는: 제출됨 → 검토 → 거부됨 ✕ → 재촬영 → 제출됨 → 검토 → 완료 ✓

진행률 바

과제 카드의 진행률 바는 완료 상태인 페이지의 비율을 나타냅니다. 모든 페이지가 완료되면 100%가 됩니다.

5 오답 노트

여러 시험에서 틀린 문제를 모아서 오답 노트를 생성할 수 있습니다. 복습에 매우 유용한 기능입니다.

오답 노트 생성 방법

오답 항목 구성

각 오답 항목에는 다음 정보가 표시됩니다:

  • 문제 번호 — 초록색 원 안에 번호 표시
  • 시험명 뱃지 — 어떤 시험에서 틀린 문제인지 표시
  • 정답 — 실제 정답 번호
  • 내 답 — 내가 선택한 답 번호

효과적인 오답 노트 활용 팁

  1. 정기적으로 생성: 시험을 볼 때마다 오답 노트를 생성하여 틀린 문제를 관리하세요.
  2. 누적 복습: 여러 시험을 함께 선택하면 누적 오답을 한눈에 볼 수 있습니다.
  3. PDF 인쇄: 오답 노트를 PDF로 다운로드하여 인쇄하면 언제든 복습할 수 있습니다.
  4. 약점 파악: 같은 유형의 문제가 반복적으로 나타나면 해당 영역을 집중 학습하세요.
💡 추천 학습법: 오답 노트를 인쇄한 뒤, 틀린 문제를 다시 풀어보고 맞았으면 체크(✓)로 표시하세요. 반복해서 맞출 때까지 계속 풀어보는 것이 가장 효과적입니다.

8 부록

자주 묻는 질문 (FAQ)

Q1. 비밀번호를 잊어버렸어요.

선생님께 비밀번호 초기화를 요청하세요. 선생님이 임시 비밀번호를 다시 설정해 주시면, 로그인 후 설정에서 새 비밀번호로 변경하면 됩니다.

Q2. 성적표가 보이지 않아요.

선생님이 성적표를 아직 등록하지 않았을 수 있습니다. 시험 후 성적표가 등록되면 "내 성적표" 메뉴에 자동으로 나타납니다. 며칠 기다려도 안 보이면 선생님께 문의하세요.

Q3. 과제 사진이 거부되었어요. 어떻게 하나요?

거부된 페이지(빨간 ✕ 표시)를 다시 눌러서 재촬영하면 됩니다. 밝은 곳에서 글씨가 선명하게 보이도록 다시 찍어 제출하세요.

Q4. 과제 마감일이 지나도 제출할 수 있나요?

기한이 지나도 제출은 가능하지만, "기한초과" 상태로 표시됩니다. 되도록 마감일 전에 제출하세요.

Q5. PDF 다운로드가 안 돼요.

브라우저의 팝업 차단이 활성화되어 있을 수 있습니다. 팝업 차단을 해제하거나 다른 브라우저(크롬 권장)를 사용해 보세요. 데이터가 완전히 로딩된 후 다운로드 버튼을 누르세요.

Q6. 모바일에서 화면이 이상하게 보여요.

브라우저를 최신 버전으로 업데이트하세요. 크롬(Chrome) 또는 사파리(Safari) 브라우저를 권장합니다. 화면이 깨질 경우 브라우저 캐시를 삭제한 후 새로고침하세요.
