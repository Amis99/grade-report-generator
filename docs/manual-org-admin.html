🚀 1. 시작하기

1.1 로그인

기관 관리자는 메인 포털(index.html)을 통해 로그인합니다. 발급받은 아이디와 비밀번호를 입력하여 접속합니다.

  1. 웹 브라우저에서 프로그램 주소에 접속합니다.
  2. 아이디(이메일 또는 사용자명)를 입력합니다.
  3. 비밀번호를 입력합니다.
  4. 로그인 버튼을 클릭합니다.
  5. 대시보드 화면으로 이동합니다.

1.3 비밀번호 변경

설정 메뉴에서 비밀번호를 변경할 수 있습니다.

  1. 좌측 사이드바에서 ⚙️ 설정을 클릭합니다.
  2. 비밀번호 변경 영역에서 현재 비밀번호를 입력합니다.
  3. 새 비밀번호와 확인 비밀번호를 입력합니다.
  4. 변경 버튼을 클릭합니다.

🏠 2. 대시보드

로그인 후 가장 먼저 표시되는 화면입니다. 현재 상태와 할 일, 최근 활동을 한눈에 확인할 수 있습니다.

주요 구성 요소

  • 환영 메시지: 관리자 이름과 오늘 날짜가 표시됩니다.
  • 할 일 피드: 진행 중인 과제, 미채점 시험 등 처리가 필요한 항목이 표시됩니다.
  • 최근 활동 타임라인: 최근 수행한 작업 이력이 시간순으로 표시됩니다.

📝 3. 시험 관리

시험을 생성하고 문제를 등록하여 관리할 수 있습니다. 본인 기관에서 생성한 시험과 국어농장 본사에서 생성한 시험 모두 목록에 표시됩니다.

✅ 안내: 본인 기관에서 생성한 시험과 국어농장 본사에서 생성한 시험이 표시됩니다. 본사 시험은 조회만 가능하며 수정/삭제할 수 없습니다.

3.1 시험 목록

시험 관리 화면에 진입하면 모든 시험이 시행일 기준 최신순으로 정렬되어 표시됩니다.

  • 검색창으로 시험명 검색이 가능합니다.
  • 시험 출처(본사/기관) 구분이 표시됩니다.
  • 시험명을 클릭하면 상세 화면으로 이동합니다.

3.2 시험 생성

  1. 시험 목록 상단의 + 새 시험 버튼을 클릭합니다.
  2. 시험명을 입력합니다. (예: "1월 모의고사")
  3. 시행일을 선택합니다.
  4. 설명(선택사항)을 입력합니다.
  5. 저장 버튼을 클릭하면 시험이 생성됩니다.
  6. 생성된 시험의 상세 화면에서 문제를 추가합니다.

3.3 시험 수정/삭제

  • 수정: 시험 목록에서 수정 버튼을 클릭하여 시험 정보를 변경합니다.
  • 삭제: 삭제 버튼을 클릭합니다. 삭제 전 확인 대화상자가 표시됩니다.
🚨 주의: 시험을 삭제하면 해당 시험의 문제, 답안, 채점 결과가 모두 삭제됩니다. 이 작업은 되돌릴 수 없습니다.

3.4 문제 추가

시험 상세 화면에서 문제를 추가합니다.

  1. 시험 상세 화면에서 + 문제 추가 버튼을 클릭합니다.
  2. 문제 추가 모달이 표시됩니다.
  3. 문제 번호, 유형, 배점 등을 입력합니다.
  4. 저장을 클릭하여 문제를 등록합니다.

3.5 문제 수정/삭제

시험 상세 화면의 문제 목록에서 각 문제 옆의 수정/삭제 버튼을 사용합니다.

  • 수정: 문제 추가와 동일한 모달이 표시되며 기존 값이 채워져 있습니다.
  • 삭제: 해당 문제와 관련된 답안 데이터도 함께 삭제됩니다.

3.6 CSV 가져오기/내보내기

CSV 가져오기 (문제 일괄 등록)

  1. 시험 상세 화면에서 CSV 가져오기 버튼을 클릭합니다.
  2. CSV 파일을 선택합니다.
  3. 미리보기에서 데이터를 확인합니다.
  4. 가져오기를 클릭하여 문제를 등록합니다.

CSV 내보내기

CSV 내보내기 버튼을 클릭하면 현재 시험의 문제가 CSV 파일로 다운로드됩니다.

✏️ 4. 답안 입력

학생들의 시험 답안을 입력하고 관리합니다.

4.1 시험 선택

  1. 사이드바에서 ✏️ 답안 입력을 클릭합니다.
  2. 상단 드롭다운에서 답안을 입력할 시험을 선택합니다.
  3. 선택한 시험의 문제 목록과 학생 목록이 표시됩니다.

4.2 학생별 답안 입력

  1. 학생 목록에서 답안을 입력할 학생을 선택합니다.
  2. 각 문제에 대한 답을 입력합니다.
  3. 객관식: 1~5 중 선택합니다.
  4. 서술형: 답안 텍스트를 입력하고, 점수를 배점 범위 내에서 입력합니다.
  5. 저장 버튼을 클릭하여 답안을 저장합니다.

4.3 CSV 일괄 업로드

여러 학생의 답안을 CSV 파일로 한 번에 업로드할 수 있습니다.

  1. CSV 업로드 버튼을 클릭합니다.
  2. 답안이 포함된 CSV 파일을 선택합니다.
  3. 미리보기에서 데이터를 확인합니다.
  4. 업로드를 클릭하여 일괄 저장합니다.

4.4 CSV 내보내기

입력된 답안 데이터를 CSV로 내보낼 수 있습니다. CSV 내보내기 버튼을 클릭하면 현재 시험의 모든 학생 답안이 다운로드됩니다.

📊 5. 채점 및 분석

시험 결과를 확인하고 문제별 분석을 수행합니다.

✅ 안내: 본인 기관 학생들의 성적만 표시됩니다.

5.2 서술형 채점

서술형 문제가 포함된 시험의 경우, 수동으로 채점해야 합니다.

  1. 채점 현황에서 서술형 미채점 항목을 확인합니다.
  2. 학생의 서술형 답안을 확인합니다.
  3. 배점 범위 내에서 점수를 입력합니다.
  4. 저장을 클릭합니다.

5.3 문제별 분석

각 문제의 정답률과 선택지 분포를 확인할 수 있습니다.

  • 정답률: 각 문제별 정답자 비율이 표시됩니다.
  • 선택지 분포: 객관식 문제의 경우 각 번호별 선택 비율이 막대그래프로 표시됩니다.
  • 정답률이 낮은 문제(30% 이하)는 빨간색으로 강조됩니다.

5.4 성적 CSV 내보내기

CSV 내보내기 버튼을 클릭하면 전체 학생의 성적 데이터가 CSV 파일로 다운로드됩니다. 이 파일에는 학생명, 총점, 각 문제별 점수 등이 포함됩니다.

📄 6. 성적표 생성

개별 학생의 성적표를 생성하여 PDF로 저장하거나 인쇄할 수 있습니다.

  1. 사이드바에서 📄 성적표 생성을 클릭합니다.
  2. 시험을 선택합니다.
  3. 학생을 선택합니다.
  4. 성적표 생성 버튼을 클릭합니다.
  5. 미리보기 화면에서 성적표 내용을 확인합니다.
  6. PDF 저장 또는 인쇄 버튼으로 출력합니다.

📎 12. 부록

12.1 자주 묻는 질문 (FAQ)

Q: 본사 시험을 수정할 수 있나요?

A: 아니요, 국어농장 본사에서 생성한 시험은 조회만 가능합니다. 문제 내용 확인과 답안 입력은 할 수 있지만, 시험 정보나 문제를 수정하거나 삭제할 수는 없습니다.

Q: 다른 기관의 학생이 보이나요?

A: 아니요, 본인 기관에 소속된 학생만 표시됩니다. 다른 기관의 학생 정보, 성적, 과제 제출 내역 등에는 접근할 수 없습니다.

Q: 관리자 메뉴가 안 보여요

A: 기관 관리자 계정은 관리자 메뉴(가입 승인, 사용자 관리)에 접근할 수 없습니다. 해당 기능은 국어농장 본사 관리자 전용입니다. 필요한 경우 본사 관리자에게 문의하세요.

Q: CSV 파일 형식은 어떻게 되나요?

A: CSV 파일은 UTF-8 인코딩으로 저장해야 합니다. 첫 번째 행은 반드시 헤더(열 이름)로 구성해야 합니다. Excel에서 저장 시 "CSV UTF-8 (쉼표로 분리)"를 선택하세요.

Q: 유사도 검사가 뭔가요?

A: AI가 학생들의 과제 제출물 간 유사도를 자동으로 분석하는 기능입니다. 높은 유사도가 감지되면 관리자가 제출물을 비교 확인하여 수동으로 통과 또는 미통과 처리할 수 있습니다.

  1. 아이디와 비밀번호를 다시 확인합니다.
